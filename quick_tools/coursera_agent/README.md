# Coursera Agent

Automation tools for Coursera course interaction.

## Components

- **coursera_agent.py** - Main automation agent
- **launch_chrome.sh** - Chrome launcher with debugging port
- **credentials.json** - API credentials (not in version control)

## Usage

```bash
# From the coursera_agent directory
python coursera_agent.py
```

## Features

- Automated browser control
- Chrome remote debugging integration
- Coursera API interaction (if applicable)

## Setup

Make sure Chrome is installed and `launch_chrome.sh` has execute permissions:

```bash
chmod +x launch_chrome.sh
```
